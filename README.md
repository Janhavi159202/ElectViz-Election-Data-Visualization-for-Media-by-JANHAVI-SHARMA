# ElectViz: Election Data Visualization for Media (Group 2)

ElectViz is a data visualization project developed as part of the Infosys Springboard program. The project focuses on analyzing and presenting election data in a clear and interactive way.

Using tools like Excel and Power BI, the dataset is cleaned, organized, and converted into visual dashboards. These dashboards help users understand important election insights such as total electors, votes casted, winning candidates, party distribution, and vote percentages.

The main objective of this project is to make election data easy to understand and visually informative for media, researchers, and the public.
